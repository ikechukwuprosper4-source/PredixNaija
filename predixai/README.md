# PredixNaija: The #1 Solana Prediction Platform for Nigerians 🏟️🇳🇬🚀

**PredixNaija** is a decentralized "Predict & Earn" platform built on **Solana**, specifically optimized for the Nigerian market. Predict on local politics, sports (NPFL/EPL), entertainment (Nollywood/Afrobeats), and the economy (Naira/USD) to earn real SOL rewards.

## 🇳🇬 Why PredixNaija?

1.  **Local Context**: AI-generated markets focused on Nigerian news, sports, and culture.
2.  **Solana Native**: Lightning-fast, low-fee transactions. Predict with $SOL and $USDC (Solana).
3.  **Predict & Earn**: Turn your local knowledge into global rewards. 
4.  **AI Oracle**: Our AI Agent verifies Nigerian news sources to resolve markets automatically.

## 🌟 Key Features

- **Naija Market Curator**: Scans Punch, Vanguard, Linda Ikeji, and X (Naija) to create trending markets.
- **Solana Wallet Integration**: Support for Phantom and Solflare.
- **Naira Tracking**: View prices and potential earnings in both SOL and Naira (NGN).

## 🛠 Tech Stack
- **Blockchain**: Solana (Mainnet-Beta / Devnet)
- **AI Brain**: Groq (Llama-3 70B)
- **Frontend**: Streamlit Marketplace
- **Data Engine**: DuckDuckGo (Free-tier search)

## 📦 Local Setup

1.  **Clone and Install**
    ```bash
    git clone <repo-url>
    cd predixai
    pip install -r requirements.txt
    ```

2.  **Configure Solana Keys**
    - `SOLANA_RPC_URL`: https://api.mainnet-beta.solana.com
    - `SOLANA_PRIVATE_KEY`: Your Solana wallet secret.
    - `GROQ_API_KEY`: For the Naija-focused AI curator.

3.  **Launch**
    ```bash
    streamlit run ui/app.py
    ```
